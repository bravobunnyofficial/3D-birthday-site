# 🎂 3D Birthday Web

A cinematic and immersive 3D birthday surprise website that makes
someone feel truly special — with a real party room, cake cutting,
confetti, sound effects, and emotional storytelling.

🌐 **Live Site** — [#3D Birthday site](https://3-d-birthday-site.vercel.app/)

---

## 🎉 About This Project

This website turns a simple birthday wish into an unforgettable
3D experience — you walk into a decorated birthday room, see a
glowing cake on a table, cut it with a knife, and watch a
cinematic slice fly toward you while the whole screen celebrates.

---

## ✨ How It Works

1. Page opens with a warm glowing birthday room
2. Hanging **Happy Birthday** banner visible with swaying letters
3. String lights and ambient glows set the celebration mood
4. User drags to explore the room — memory frames on the walls
5. User clicks the knife on the table
6. Knife animates and cuts the cake cinematically
7. Cake slice flies toward the screen, spins 180°, then vanishes
8. Confetti explodes and **Happy Birthday Khushi!** appears

---

## 🎯 Features

- 🏠 Immersive 3D birthday room with realistic lighting
- 🎈 Hanging letter banner — H-A-P-P-Y B-I-R-T-H-D-A-Y
- 💡 Glowing string lights with flicker animation
- 🖼️ Memory photo frames on room walls
- 🎂 Realistic GLB cake model on a wooden table
- 🔪 Interactive knife — click to cut the cake
- 🍰 Cinematic cake slice flies toward screen + 180° spin
- 🎉 Confetti explosion on cake cut
- 🔊 Sound effects — knife cut + celebration melody + clapping
- 🖱️ Drag to rotate room + scroll to zoom
- 📱 Fully mobile responsive with touch support
